# 📝 راهنمای تمپلیت مقاله - دیتابــان

این راهنما برای ایجاد و ویرایش مقالات جدید در سایت دیتابــان طراحی شده است.

## 📁 ساختار فایل‌ها

```
data/articles/
├── article-template.html    # تمپلیت اصلی (کپی کنید برای مقاله جدید)
├── article-1.html          # مقاله ۱
├── article-2.html          # مقاله ۲
├── article-N.html          # مقالات بعدی
└── README.md               # این فایل
```

## 🚀 ایجاد مقاله جدید

### مرحله ۱: کپی تمپلیت
```bash
# کپی کردن تمپلیت
cp article-template.html article-N.html
```

### مرحله ۲: ویرایش متادیتا

در تگ `<article>` موارد زیر را تغییر دهید:

```html
<article 
    data-id="ARTICLE_ID"              <!-- شماره مقاله (عدد) -->
    data-category="CATEGORY"          <!-- دسته‌بندی: تکنولوژی، امنیت، سیاست، اقتصاد -->
    data-author="AUTHOR"              <!-- نام نویسنده -->
    data-date="DATE"                  <!-- تاریخ به فارسی: ۱۸ آبان ۱۴۰۳ -->
    data-reading-time="READING_TIME"  <!-- زمان مطالعه به دقیقه (عدد) -->
    data-excerpt="EXCERPT"            <!-- خلاصه کوتاه مقاله -->
    data-tags="TAG1,TAG2,TAG3"        <!-- برچسب‌ها با کاما جدا شوند -->
    data-featured="true">             <!-- مقاله ویژه: true یا false -->
```

**مثال:**
```html
<article 
    data-id="3"
    data-category="تکنولوژی"
    data-author="تیم دیتابــان"
    data-date="۲۰ آبان ۱۴۰۳"
    data-reading-time="15"
    data-excerpt="بررسی تأثیر هوش مصنوعی بر آینده کار"
    data-tags="هوش مصنوعی,تکنولوژی,آینده"
    data-featured="true">
```

### مرحله ۳: ویرایش عنوان صفحه

```html
<title>عنوان مقاله - دیتابــان</title>
```

### مرحله ۴: تنظیم تصویر کاور

```html
<img class="article-image" src="../../assets/images/IMAGE_NAME.png" alt="ALT_TEXT">
```

**نکات مهم:**
- تصویر را در پوشه `assets/images/` قرار دهید
- نام فایل را به انگلیسی و بدون فاصله بنویسید
- فرمت‌های مجاز: PNG, JPG, WebP
- سایز توصیه شده: 1200x600 پیکسل

### مرحله ۵: ویرایش محتوای مقاله

#### عنوان اصلی
```html
<h1 class="article-body-title">عنوان اصلی مقاله</h1>
```

#### خلاصه مقاله (Lede)
```html
<p class="article-lede">خلاصه کوتاه مقاله که در ابتدا نمایش داده می‌شود.</p>
```

#### پاراگراف‌ها
```html
<p>متن پاراگراف...</p>
```

#### عناوین فرعی
```html
<h2 id="section-id">عنوان بخش اصلی</h2>
<h3>زیرعنوان</h3>
```

**نکته:** از `id` برای لینک‌دهی به بخش‌های مختلف استفاده کنید.

#### لیست‌ها
```html
<!-- لیست نقطه‌ای -->
<ul>
    <li>مورد اول</li>
    <li>مورد دوم</li>
    <li>مورد سوم</li>
</ul>

<!-- لیست شماره‌دار -->
<ol>
    <li>مرحله اول</li>
    <li>مرحله دوم</li>
    <li>مرحله سوم</li>
</ol>
```

#### نقل قول (Blockquote)
```html
<blockquote>
متن نقل قول یا نکته مهم که می‌خواهید برجسته شود.
</blockquote>
```

#### تصاویر داخل متن
```html
<img src="../assets/images/example.png" alt="توضیح تصویر" 
     style="width: 100%; border-radius: 8px; margin: 1.5rem 0;">
<p style="text-align: center; color: var(--text-secondary); font-size: 0.9rem; 
          margin-top: -1rem; margin-bottom: 2rem;">
    توضیح زیر تصویر
</p>
```

### مرحله ۶: تنظیم برچسب‌ها

در انتهای مقاله، برچسب‌ها را ویرایش کنید:

```html
<div class="article-tags">
    <h3>برچسب‌ها</h3>
    <div class="tags-list">
        <a href="#" class="tag">برچسب۱</a>
        <a href="#" class="tag">برچسب۲</a>
        <a href="#" class="tag">برچسب۳</a>
    </div>
</div>
```

**نکته:** برچسب‌ها باید با `data-tags` در متادیتا همخوانی داشته باشند.

### مرحله ۷: تنظیم مقالات مرتبط

در سایدبار، لینک مقالات مرتبط را تغییر دهید:

```html
<a href="article-X.html" class="related-item">
    <span class="related-item-number">۱</span>
    <div class="related-item-content">
        <span class="related-item-title">عنوان مقاله مرتبط</span>
    </div>
</a>
```

## 🎨 المان‌های استایل

### کلاس‌های مهم

| کلاس | کاربرد |
|------|--------|
| `.article-body-title` | عنوان اصلی مقاله |
| `.article-lede` | خلاصه مقاله |
| `.article-body-content` | محتوای اصلی |
| `.article-tags` | بخش برچسب‌ها |
| `.tag` | هر برچسب |
| `.share-btn` | دکمه‌های اشتراک‌گذاری |
| `.related-item` | مقاله مرتبط |

### متغیرهای CSS

```css
--text-primary: رنگ متن اصلی
--text-secondary: رنگ متن فرعی
--text-muted: رنگ متن کم‌رنگ
--primary-color: رنگ اصلی سایت (قرمز)
--card-bg: پس‌زمینه کارت‌ها
--dark-bg: پس‌زمینه تیره
--border-color: رنگ حاشیه‌ها
--radius-sm: شعاع گوشه کوچک
--radius-md: شعاع گوشه متوسط
--radius-lg: شعاع گوشه بزرگ
```

## 📱 Responsive Design

تمپلیت به صورت خودکار برای سایزهای مختلف صفحه بهینه می‌شود:

- **Desktop** (>1024px): دو ستون (محتوا + سایدبار)
- **Tablet** (768-1024px): دو ستون با فاصله کمتر
- **Mobile** (<768px): تک ستون (محتوا بالا، سایدبار پایین)

## 💡 نکات مهم

### ۱. استفاده از تصاویر
- همیشه `alt` برای تصاویر بنویسید
- از تصاویر بهینه شده استفاده کنید (WebP ترجیحاً)
- سایز فایل تصاویر را کم نگه دارید (<500KB)

### ۲. سئو (SEO)
- عنوان مقاله باید منحصر به فرد باشد
- خلاصه مقاله (`data-excerpt`) را دقیق بنویسید
- از برچسب‌های مرتبط استفاده کنید
- از `id` برای عناوین استفاده کنید

### ۳. دسترسی‌پذیری
- از عناوین به ترتیب استفاده کنید (h2, h3, h4)
- متن جایگزین (`alt`) برای تصاویر بنویسید
- کنتراست رنگ‌ها را رعایت کنید

### ۴. عملکرد
- از تصاویر lazy-load استفاده کنید
- فایل‌های CSS و JS را minify کنید
- از CDN برای منابع استاتیک استفاده کنید

## 🐛 رفع مشکلات رایج

### مشکل: تصویر نمایش داده نمی‌شود
**راه‌حل:** مسیر فایل را بررسی کنید. باید `../../assets/images/` باشد.

### مشکل: استایل‌ها اعمال نمی‌شوند
**راه‌حل:** لینک‌های CSS را بررسی کنید:
```html
<link rel="stylesheet" href="../../css/main.css">
<link rel="stylesheet" href="../../css/article.css">
<link rel="stylesheet" href="../../css/article-template.css">
```

### مشکل: سایدبار sticky نیست
**راه‌حل:** مطمئن شوید کلاس `article-sidebar-right` درست است.

### مشکل: برچسب‌ها # ندارند
**راه‌حل:** استایل CSS به صورت خودکار # اضافه می‌کند. نیازی به نوشتن دستی نیست.
